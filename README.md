# Todoアプリ（Docker × Flask × MySQL × AWS EC2）

## 概要
Dockerを用いてFlaskアプリケーションを構築し、AWS EC2上にデプロイしたTODOアプリです。  
タスクの追加・削除・完了機能・フィルター機能を実装しています。

---

## 使用技術
- Python（Flask）
- MySQL
- Docker / Docker Compose
- AWS EC2

---

## 機能一覧
- タスク追加
- タスク削除
- 完了チェック機能
- フィルター機能（全て / 未完了 / 完了）
